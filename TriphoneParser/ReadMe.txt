========================================================================
    CONSOLE APPLICATION : TriphoneParser Project Overview
========================================================================

This project implement reducing acustic model size via removing unused triphones:

INPUT:

-hmm  "acustic-model-dir-name"
-dict "phonetic dictionary"
-s	  "file with all possible sentences. one per line"

OPTIONAL:

-dir  "output dir name"


OUTPUT:

New acustic model without unused triphones in output directory
